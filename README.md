# How-To-Implement-SCRUM

SCRUM is a methodology that has some pre-defined ceremonies. If done right, it has immense benifits, some of them are as follows:

1. Binds the effort of the entire team towards a common goal.
2. Provides transaprency to stakeholders.
3. Demands accountability from people doing the job. 
4. Helps to estimate timelines & future risks.
5. Places checks and balances if things are not going as per plan.
